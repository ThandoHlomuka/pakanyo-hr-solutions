# Pakanyo HR Solutions

A responsive business website for Pakanyo HR Solutions — a South African HR and skills development company specializing in building workforce capability, leadership effectiveness, and high-performing teams.

## Pages

- **Home** — Hero section, core solutions, why choose us, call to action
- **About** — Company story, vision, mission, approach
- **Services** — Occupational qualifications, skills programmes, recruitment, team performance, learning material development
- **Programmes** — Leadership development, team performance, workplace effectiveness, HR & recruitment, customised solutions
- **Contact** — Enquiry form and company contact details

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (IntersectionObserver, scroll effects)
- Vercel (deployment)

## Features

- Responsive design (mobile, tablet, desktop)
- Interactive splash onboarding animation with HR workspace scene
- Scroll-triggered fade-in animations
- Smooth navigation with mobile hamburger menu
- Cross-browser compatible (Chrome, Firefox, Safari, Edge, Huawei)

## Deployment

Auto-deploys to Vercel on push to `master` branch.
